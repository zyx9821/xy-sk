📖 系统部署与对接使用说明

架构说明： 本系统运行在 Cloudflare Workers Serverless 环境下。完全免疫 Ddos，且无需购买任何服务器即可实现秒级波场与 EVM 多链 USDT 收款监听。

1. 靓号地址添加规则

在【基础配置】的地址池中：
- 只要你输入以 T 开头的地址，系统将自动调用 TRON (波场) 引擎进行扫描。
- 只要你输入以 0x 开头的地址，系统将自动判定为 EVM 兼容链，并同时调用 ETH、BSC、AVAX 三个引擎对该地址进行并发扫描。无需手动指定网络！

2. 定时轮询 (Cron) 配置须知

系统默认通过 wrangler.toml 配置了每分钟一次的自动巡检。如果在 Cloudflare 仪表盘没有生效，请前往 Workers -> 设置 -> 触发器 (Triggers) 中手动添加 Cron 表达式：* * * * *。

3. 前端发卡网 PHP 回调验证代码

当系统查到账单后，会向你在【回调路由】中配置的 URL 发送 POST 数据包。由于我们支持了全链环境，为了防止重放攻击，签名校验必须包含 network 字段。请将以下代码集成到你的发卡网中处理回调：

<?php // 接收监控矩阵发来的原始 JSON 数据 $jsonStr = file_get_contents('php://input'); $post = json_decode($jsonStr, true); // 1. 获取基础数据 $network = $post['network']; // TRON / BSC / ETH / AVAX $tx_hash = $post['tx_hash']; // 交易哈希，请作为唯一索引防重复发卡 $amount = $post['amount']; // 实际到账的 USDT 金额 $sign = $post['sign']; // 签名密文 // 2. 验证签名 (防篡改核心逻辑) // 拼接规则：网络标识 + 交易哈希 + 金额 + 你在面板设置的APP_SECRET $my_secret = "这里填你面板里的APP_SECRET"; $calc_sign = hash('sha256', $network . $tx_hash . $amount . $my_secret); if ($calc_sign !== $sign) { die(json_encode(["code" => 403, "msg" => "签名校验失败！"])); } // 3. 校验通过，执行发卡网发货逻辑 // TODO: 去数据库查询对应金额的未支付订单，并进行卡密发放 // ... echo json_encode(["code" => 200, "msg" => "回调处理成功"]); ?>
